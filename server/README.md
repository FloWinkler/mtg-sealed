# MTG Multiplayer Server

## Starten

```bash
cd server
node index.js
```

Der Server läuft standardmäßig auf Port 4000.

## Features (MVP)
- Login per E-Mail
- Lobby für 2 Spieler
- Ready-Status und Set-Auswahl
- Startsignal für Deckbuilding
